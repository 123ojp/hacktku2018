# 2018 淡江黑客松
## 活動資料
1. 題目:讓科技不再冷漠
2. 組名:阿肥小公主
3. 費時:30小時

## 啟動方式
1. 執行 chat.py (websocket port:8080)
2. 執行 makequs.py (db控制)
3. 執行 manage.py (Django port:8000)

## requirement
1. Python3
2. Django
3. Flask
4. Gevent-websocket
5. Requests
